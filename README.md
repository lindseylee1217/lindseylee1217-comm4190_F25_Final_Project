# Looking Glass

**An LLM-Powered Journal Reflection Guide**

Authors: Hanxi Guo & Lindsey Lee  
Course: COMM4190 Final Project

---

## Project Overview

Looking Glass is an intelligent journal analysis tool that guides user reflection based on journal entries using LLM technology. The system can:

- Identify tones, emotions, and recurring topics in each journal entry
- Track how entries shift over time
- Provide personalized reflection guidance
